Olá, eu sou Thiago Gonçalves Santos.

Sou um desenvolvedor brasileiro, iniciei minha jornada na programação em 2017 quando iniciei meus estudos no ensino médio na instituição de ensino Etec, onde o ensino médio é realizado juntamente com um curso técnico, onde na época optei por ser o curso técnico em computação. Após terminar o ensino médio em 2019, no ano seguinte iniciei a faculdade, no mesmo ano de 2020, em outubro, consegui minha primeira oportunidade de emprego na área com um estágio na empresa Followize.

Na Followize comecei com um estágio que durou 6 meses, após esse período fui efetivado e me tornei desenvolvedor jr I. O ano de 2022 marcou o fim da minha faculdade e com isso atualmente continuo trabalhando na mesma instituição porém ela mudou de nome, o antigo Followize agora se chama DealerSpace.

Habilidades

Php

Linguagem de progração criada especificamente para internet, lidera o mercado web por anos, foi inovado por ser a primeira voltada para web e até hoje segue se atualizando. Tive meu primeiro contato com o Php quanto fui fazer meu tcc para o curso tecnico, desde então virou minha principal linguagem de programação e estudo ela a 6 anos.

Laravel

Laravel é o framework mais conhecido para Php e o mais usado no mercado, ele visa deixar o desenvolvimento mais rapido e simplificado, com otimas soluções já desenvolvidas e permitindo a escalabilidade dos projetos. Tive o primeiro contato no projeto de estágio que participei.

Git

Git é um sistema de versionamento de codigo mais utilizado e o padrão do mercado. Começei a utilizar e estudar ele quando fui desenvolver meu Tcc para o curso tecnico, na busca de conseguir salvar o codigo e facilitar que mais pessoas consiga trabalhar no mesmo projeto ao mesmo tempo.

Postgresql

O PostgreSql é um dos bancos de dados mais robusto que temos no mercado e é o mais indicado para se utilizar junto com o Php pois os dois compoem uma das stacks mais populares para web. Tive o primeiro contato durante o projeto de estagio pois fazia parte da stack utilizada no sistema.

Sql

Sql é uma linguagem de consulto de banco de dados desenvolvida pela IBM em 1970, que ajuda no gerenciamento de bancos de dados relacionados e virou padräo do mercado, sendo até representada como um tipo de linguagem de banco de dados. Foi a primeira linguem voltada para banco de dos que eu aprendi no curso tecnico.

Javascript

Javascript é uma linguaguem de programação voltada para o desenvolvimento web, inicialmente foi desenvolvida para permiter que os desenvolvedores tenham mais possibilidades para criar funções na interface e que fosse interpretada pelo proprio navegador sem precisar enviar para um servidor.

Typescript

Typescript é uma linguaguem de programação baseada no Javascript, que possui uma sintaxe mais robusta e complexa, permitindo melhor visualização do codigo como a possibilidade de tipar as variaveis.

Angular

Angular é um framework front-end de Javascript/Typescript (é opicional qual linguaguem vai utilizar), como framework ele tem a função de fornecer uma estrutura base que auxilia e ajuda o desenvolvedor nas suas criações, fazendo ele não perder tempo com a parte de escrutura e funcionalidades muito utilizadas.

Scrum

Scrum é um framework de gerenciamento agil de projetos, muito utilizado no universo de desenvolvimento, onde se baseia em uma premissa básica onde ao invez de tentar finalizar o projeto por inteiro que é um processo muito demorado e grande, dividir ele em pequenas entregas e por ondem de prioridade, conseguindo assim tornar o projeto funcional em menos tempo e melhorar ele gradualmente.
